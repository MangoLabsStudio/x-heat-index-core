# X Heat Index Core

Public, sanitized collector and aggregation core for X/Twitter campaign heat signals.

This export intentionally excludes private deployment documents, hosted API routes, pool/share allocation code, customer campaign data, raw graph artifacts, production secrets, and original repository history.

## Included

- X/Twitter collection scripts
- Campaign graph normalization helpers
- Hourly attention aggregation logic
- Health/validation utilities

## Excluded

- Real campaign/pool data
- Pool/share allocation APIs and operational docs
- Customer names, private campaign ids, and KOL lists
- Production host paths, Railway URLs, API keys, database URLs, and tokens
